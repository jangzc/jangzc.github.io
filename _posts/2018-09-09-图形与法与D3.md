## 1. 图形语法的流程 与 D3链式API 很合适的进行了对应<br>
D3的设计决定中，最重要的一点是把"图形语法"的理念加入到了D3的设计中

## 2. D3比图形语法低一个级别，并且是图形语法的上一个级别，因此D3通常被误认为是绘图库

## 3. 图形语法最基本的三层<br>
+ Data
    + The actual variables to be plotted
+ Aesthetics
    + The scales onto which we will map out data
+ Geometries
    + Shapes used to represent out data

<img src="https://www.science-craft.com/wp-content/uploads/2014/06/ggplot-1.png"/>

## 4. 更多的细节<br>
在基本三层的基础上，我们可以对plot的细节进行修改，为了是更清晰的表达出信息的意思。我们可以定义更多的功能:<br>
+ Facets
    + Rows and columns of sub-plots
+ Statistics
    + Statistical modes & summaries
+ Coordinates
    + The plotting space we are using.
    
<img src="https://www.science-craft.com/wp-content/uploads/2014/06/ggplot-2.png"/>

## 5. 调整设计元素<br>
+ Theme
    + Describes non-data ink
